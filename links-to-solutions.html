Reliability on top of UDP

This document contains links to the solutions to the tasks.

How to run the solutions

To run the solutions you need to have the Rust toolchain installed.

To start the server, run:

cargo run --bin server

To start the client, run:

cargo run --bin client

Solutions

There are two links in front of each task description. The examine link points to the Github commit containing the solution. The download link points to the release page of the solution from where you can download the source code.

Virtual Socket

  1. [Examine] [Download] (2p) Make an application for testing purposes, e.g. client sends data to server and server displays it on screen. It is enough for the application to send data only in one direction (and later acknowledgements in the other). Use the test application to test what you implement later.
  2. [Examine] [Download] (2p) Make a virtual socket that randomly drops a packet (i.e. does not pass it to UDP socket or to the application)
  3. [Examine] [Download] (3p) Add to the virtual socket: randomly delays a packet, before passing it forward
  4. [Examine] [Download] (5p) Add to the virtual socket: randomly generates bit errors on the data of the packet, a single bit error is enough.
